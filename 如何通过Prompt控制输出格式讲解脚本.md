# 如何通过Prompt控制输出格式（JSON、Markdown）讲解脚本

**时长：约5-6分钟 | 字数：约950字**

---

## 【开场白】（约30秒）

大家好！今天我们来讨论一个非常实用的技巧——**如何通过Prompt控制输出格式**。

当你需要AI的输出能被程序自动处理时，格式控制就变得至关重要。无论是JSON、Markdown还是其他格式，掌握这个技巧都能大幅提升工作效率。

---

## 【为什么需要控制输出格式】（约1分钟）

首先，为什么需要明确要求输出格式？

**原因一：程序自动处理**

当输出是JSON格式时，程序可以直接解析和使用。如果是随意的文本格式，需要手动提取和清理。

**原因二：确保一致性**

明确要求格式能确保每次的输出都遵循相同的结构。这对批量处理多个请求特别重要。

**原因三：减少后处理工作**

格式正确的输出可以直接使用，而无需额外的整理工作。

---

## 【JSON格式控制】（约1.5分钟）

JSON是最常用的结构化格式。怎样让AI输出JSON？

**方法一：直接指定**

最简单的方式就是直接告诉AI：

> "请用JSON格式输出以下信息"

**方法二：给出示例格式**

提供一个你期望的JSON结构示例：

> "请用以下JSON格式输出：
> {
>   "name": "姓名",
>   "age": 年龄,
>   "city": "城市"
> }"

**方法三：指定字段和类型**

更详细地说明每个字段：

> "请输出JSON对象，包含以下字段：
> - name (string): 人名
> - age (number): 年龄
> - skills (array): 技能列表"

**实际例子：**

问题：请提取这段文字中的重要信息并用JSON格式输出。

文字：小王是一位32岁的Python开发者，住在北京，会Java和C++。

AI的输出应该是：
```json
{
  "name": "小王",
  "age": 32,
  "profession": "Python开发者",
  "location": "北京",
  "skills": ["Java", "C++"]
}
```

## 【Markdown格式控制】（约1.5分钟）

Markdown是易读且结构化的格式。怎样控制Markdown输出？

**方法一：要求使用标题和列表**

> "请用Markdown格式回答，使用# 和 ## 作为标题，用 - 作为列表项"

**方法二：给出模板**

```markdown
# 主题
## 第一部分
内容

## 第二部分
内容
```

**方法三：指定具体结构**

> "请用以下Markdown结构：
> # 标题
> ## 简介（2-3句）
> ## 关键点（用列表，3-5项）
> ## 结论"

**实际例子：**

问题：用Markdown格式介绍Python语言。

AI的输出应该是：
```markdown
# Python语言介绍

## 简介
Python是一种高级、通用的编程语言，以易读易学而著称。

## 主要特点
- 简洁的语法
- 强大的库生态
- 广泛的应用领域
- 活跃的社区

## 应用领域
- 数据分析和科学计算
- 网络开发
- 人工智能和机器学习
```

## 【其他常见格式】（约1分钟）

除了JSON和Markdown，还有其他格式：

**CSV格式**

> "请用CSV格式输出，用逗号分隔，第一行是列标题"

输出：
```
Name,Age,City
Tom,25,Shanghai
Lucy,28,Beijing
```

**YAML格式**

> "请用YAML格式输出"

输出：
```yaml
person:
  name: Tom
  age: 25
  city: Shanghai
```

**Table格式**

> "请用Markdown表格格式输出"

输出：
```
| Name | Age | City |
|------|-----|------|
| Tom  | 25  | Shanghai |
| Lucy | 28  | Beijing |
```

## 【高级技巧】（约1分钟）

如何确保AI严格遵循格式？

**技巧一：提供完整示例**

给一个完整的示例而不仅仅是一个骨架。AI会更准确地模仿。

**技巧二：指定冗余和精度**

> "JSON中所有数值必须使用整数，不允许null值"

**技巧三：验证语句**

> "请确保输出是有效的JSON，可以被JSON解析器解析"

**技巧四：分步指令**

> "1) 收集信息
> 2) 组织成结构
> 3) 转换为JSON格式
> 4) 验证有效性"

---

## 【常见问题】（约45秒）

**问题1：AI不遵循格式怎么办？**

答：可能需要更明确的示例或在系统提示中强调格式要求。

**问题2：复杂嵌套结构怎样指定？**

答：提供一个完整的示例，包括所有可能的嵌套级别。

**问题3：格式优先还是内容优先？**

答：通常应该说清楚，但最好是"既要格式正确，也要内容正确"。

---

## 【总结】（约30秒）

**通过明确的Prompt指导，可以让AI输出指定的格式。**

关键方法：
- 直接说明格式类型
- 提供完整示例
- 指定每个字段的类型和含义
- 在必要时请求验证

掌握这个技巧，你就能自动化处理AI的输出，大幅提升工作效率。
