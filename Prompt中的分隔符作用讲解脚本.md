# Prompt中的"分隔符"作用讲解脚本

## 【开场白】（约30秒）

大家好！今天我们来讨论一个看起来很小但作用很大的技巧——**Prompt中的分隔符**。

分隔符是什么？就是用来分开不同内容块的符号或文本。虽然看起来很基础，但正确使用分隔符能显著改善AI的理解和输出质量。

---

## 【什么是分隔符】（约1分钟）

**分隔符就是用来清晰地分隔Prompt中不同部分的符号或标记。**

常见的分隔符包括：
- `###` 或 `---`：用于分隔不同的段落或指令
- `""` 或 `''`：用于标记具体的内容或数据
- `###输入：` 和 `###输出：`：用于标记输入和输出
- 括号 `[]` 或 `{}`：用于标记特殊内容
- 空行：用于视觉分隔

为什么使用分隔符？因为它能帮助AI明确地区分：
- 指令和内容
- 示例和实际任务
- 输入和期望输出
- 不同的指令块

---

## 【为什么分隔符重要】（约1分钟）

分隔符看起来很简单，但为什么重要？

**原因一：减少混淆**

没有明确的分隔，AI可能会混淆指令和内容。比如：

模糊的Prompt：
> 请总结这段文字：这是第一行。这是第二行。请输出结果。

AI可能会把"请输出结果"当做内容的一部分。

有分隔符的Prompt：
> 请总结以下文字：
> ###输入开始###
> 这是第一行。这是第二行。
> ###输入结束###
> 
> ###输出格式###
> 总结结果

现在AI就很清楚各个部分的边界。

**原因二：提升准确性**

研究表明，使用分隔符的Prompt在复杂任务上准确率能提升15-25%。

**原因三：便于扩展**

当你需要添加更多内容时，有分隔符的Prompt更容易修改和扩展。

## 【常见的分隔符使用方式】（约1.5分钟）

让我给你展示几种有效的分隔符使用方式。

**方式一：三个井号分隔**

```
### 指令 ###
请总结以下文章。

### 文章内容 ###
[文章文本]

### 输出格式 ###
用3句话总结
```

**方式二：使用角括号**

```
<instruction>
请分析这段代码的性能问题
</instruction>

<code>
[代码内容]
</code>

<output_format>
1. 性能问题列表
2. 每个问题的解决方案
</output_format>
```

**方式三：使用双引号标记内容**

```
请回答这个问题："用户反复收不到邮件该怎么办？"

示例答案1："首先检查垃圾邮件文件夹"
示例答案2："确认邮件地址输入正确"

请基于这些示例生成5个类似的常见问题和答案。
```

**方式四：简洁的横线分隔**

```
任务：生成一个营销文案

---

目标产品：笔记本电脑

目标受众：学生

---

要求：100字以内，使用轻松的语调

---
```

## 【分隔符的最佳实践】（约1.5分钟）

如何有效地使用分隔符？

**实践一：保持一致性**

选择一种分隔符风格并在整个Prompt中保持一致。混用不同的分隔符会造成混淆。

好的做法：
```
### 指令 ###
...
### 内容 ###
...
### 格式 ###
...
```

**实践二：嵌套结构时使用层级**

当有多个层级的内容时，使用不同的分隔符表示层级：

```
### 主任务 ###

内容说明

#### 子任务1 ####
...

#### 子任务2 ####
...

###输出要求###
```

**实践三：引号标记用户输入**

```
用户输入："我想学习编程"

请基于这个输入生成一个学习计划。
```

**实践四：使用描述性的分隔标签**

代替单纯的符号，使用描述性的标签更清楚：

```
---【系统指令开始】---
你是一个专业的AI助手
---【系统指令结束】---

---【用户任务开始】---
请为我写一个邮件
---【用户任务结束】---
```

## 【分隔符的影响】（约1分钟）

正确使用分隔符有什么实际影响？

**影响一：提升理解准确性**

有分隔符的Prompt，AI对不同部分的理解会更准确。

**影响二：减少遗漏**

AI不太可能遗漏被明确分隔出来的指令。

**影响三：便于调试**

当输出不符合预期时，分隔符清晰的结构使调试更容易。

**影响四：可读性提升**

不仅AI能理解得更好，人类阅读时也更容易。

## 【总结】（约30秒）

**分隔符是一个小技巧，但对Prompt的效果有显著影响。**

关键要点：
- 使用分隔符清晰地分隔不同部分
- 保持分隔符的一致性
- 对复杂Prompt使用嵌套的分隔结构
- 使用描述性标签提升清晰度

这是一个简单但高效的优化技巧，值得在每个Prompt中应用。
